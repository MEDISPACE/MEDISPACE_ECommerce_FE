# About MediSpace E-Commerce Frontend

MediSpace E-Commerce Frontend is the web interface for a digital pharmacy platform. It delivers the customer storefront, account portal, pharmacist workspace, and admin console in a single React Router application.

The application supports pharmaceutical product discovery, cart and checkout, prescription upload with OCR-assisted review, pharmacist consultation, AI chat, loyalty and coupons, HealthHub content, community discussion, and LiveKit-powered video health events.

This frontend is designed to work with the MediSpace Node.js backend and Python AI services, including OCR, recommendation, and chat AI services.
